# aner_webview

一个仅支持单 URL 的 GUI 浏览器. 为了方便让一些指定人群快速访问网站, 如公司员工访问公司的IO管理后台.

#### 使用第三方库

- PyQt5
- PyQtWebEngine
- pyinstaller (打包工具)

#### 使用说明

在 `__init__.py` 文件中设置 URL 地址, 窗口标题等. 然后执行 `index.py` 文件即可.

#### 打包命令说明

###### 打包单执行文件命令
```
pyinstaller -F index.py --noconsole -n xxx --clean
```

#### 当前问题

本项目最主要的应用是打包后通过各种途径传输给使用者, 让使用者通过执行文件直接访问目标 URL, 但是此项目所用到的 PyQt5 和 PyQtWebEngine 两个库都特别大(前者58M左右, 后者 46M 左右), 若打包为单执行文件, 则尺寸大小直接超出100M, 最常用的微信最大支持的传输大小仅100M, 造成了不便. 目前还没有什么解决方案.
